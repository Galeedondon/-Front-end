# HTML基礎語法
|    HTML   |      中文     |
|-----------|:-------------:|
|\<\!DOCTYPE html\>| 定義文件為HTML5 |
|\<html\>|敘述該元件是一個HTML頁面的根元素|
|\<head\>|敘述該元素包含有關文檔的元信息|
|\<title\>|敘述該元素指定該文檔的標題|
|\<body\>|敘述該元素包含可見的頁面內容|
|\<h1\>|標題(Heading)，大小為24px|
|\<h2\>|標題(Heading)，大小為22px|
|\<h3\>|標題(Heading)，大小為18px|
|\<h4\>|標題(Heading)，大小為16px|
|\<h5\>|標題(Heading)，大小為12px|
|<h6>|標題(Heading)，大小為10px|
