# Grid
> 簡單來說Bootstrap的布局(grid)分為超小型、小型、中型、大型或超大型設備大小，而在以上裝置的布局都分為十二等分。   
> 以下圖舉例，滿版也就是12/12，bootstrap語法中就是col-*-12 *** *號為對應裝置 ***，換句話說螢幕的25%等於1/4，也就是col-*-3。
